Git

Que es Git y para que sirve?

Git es una herramienta creada por Linus Torvalds que te ayuda a controlar las versiones de un proyecto, el cual guarda un historial de los cambios realizados. Algunos beneficios que no brinda Git son los siguientes :

  • Repositorio distribuido
  • No requiere conexión a Internet
  • Integridad de datos
  • Fomenta la colaboración debido al Open Source

Como instalar Git?

En Windows

Para instalar git en Windows puedes usar el instalador que se descarga en la página git.scm.com

En Mac

Para instalar git en macOS existen 2 opciones. En primer lugar, usar la interfaz gráfica del instalador de la página sourceforge.net. Por otro lado, puedes instalar homebrew para tener disponible el siguiente comando :

$ brew install git

En Linux

Para instalar Git en un Sistema Operativo basado en Linux, deberás abrir la terminal y colocar el siguiente comando :

sudo apt-get install git

Los comandos básicos en git

Existen tres secciones principales en un proyecto con Git, en los cuales se gestionan los cambios :

  • Working Directory
  • Staging Área
  • Git Directory o Repositorio

Iniciar git en un proyecto - Git init

El comando "git init" sirve iniciar git en un repositorio o proyecto nuevo.

Agregar al stage - Git Add

Un commit es cada uno de los cambios realizados en el historial de git.

Para agregar los archivos con los cambios deberá colocar el siguiente comando :

git add .

Agregar al working - Git Commit

Un commit es cada uno de los cambios realizados en el historial de git.

Para agregar un commit deberá colocar el siguiente comando :

git commit -m "Este es mi primer commit"
